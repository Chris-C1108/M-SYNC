iOS快捷指令配置指南

按照以下步骤在iOS快捷指令中配置M-SYNC：

  1. 创建一个"iOS快捷指令"类型的Token
  2. 复制生成的Token
  3. 在iOS设备上打开"快捷指令"应用
  4. 创建新的快捷指令，添加"获取网页内容"操作
  5. 配置以下参数：

快捷指令配置：

URL: http://localhost:3000/api/v1/messages/publish
方法: POST
请求体: JSON
{
  "token": "您的Token",
  "messageType": "TEXT",
  "content": "剪贴板内容或输入文本"
}
